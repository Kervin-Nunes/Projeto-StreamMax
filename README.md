# StreamMax — Landing Page 🎬

Projeto de landing page para um serviço fictício de streaming, inspirado em plataformas como Netflix. A página apresenta seções de apresentação (hero), diferenciais do serviço, planos de assinatura (Básico, Padrão e Premium) e formulário de contato.

**Tecnologias utilizadas:** HTML5 · CSS3 · JavaScript
